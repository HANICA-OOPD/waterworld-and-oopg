# Oefensuggesties
- [ ] 1. Zorg dat bellen zodra de speler ze aanraakt roze worden.
- [ ] 2. Zorg dat er twee zwaardvissen rondzwemmen (de 2e met een andere snelheid dan de eerste).
- [ ] 3. Introduceer een 2e tiletype (even groot als de 1e) waar bellen niet doorheen kunnen (als ze daar tegenaan komen, blijven ze dus hangen).
- [ ] 4. Zorg dat de speler naar een willekeurige plek in de wereld teleporteert als-ie een zwaardvis aanraakt.
- [ ] 5. Maak er een spelletje van zodat de speler bellen kan vangen:
   - A voor elke bel krijgt de speler punten
   - B voor elke keer dat de speler de/een zwaardvis raakt raakt hij punten kwijt
   - C de score staat bovenin beeld (in plaats van het aantal geknapte bellen) 
   - D het spel slaat de highscore op bij afsluiten (in plaats van het aantal geklapte bellen)
